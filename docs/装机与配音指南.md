# 装机与配音指南（produce 真出片 + F5 克隆音）

> 当前沙箱 venv 是空的；本文是在**你自己的开发机**上把 `produce` 跑通的清单。

---

## A. produce 真出片依赖（按顺序装）

```bash
# 1. ffmpeg（系统级，视频合成/音频转码必须）
brew install ffmpeg                 # macOS

# 2. Python 依赖（whisper 字幕 / edge-tts / 智谱绘图 / 语义去重等）
pip install -r requirements.txt

# 3. 字体（字幕/封面用）：macOS 自带 Arial Unicode MS；
#    若缺字或想换风格，把 .env 的 FONT_PATH 指向思源黑体等中文字体文件

# 4.（可选）仅采集/RPA 才需要
playwright install chromium
```

跑通验证：
```bash
python scripts/main.py generate 2     # 先有素材（只需 LLM key）
python scripts/main.py setup          # 账号矩阵
python scripts/main.py produce        # 出片，看 output/日期/主题/ 下：
                                      #   viral_output.mp4 / cover_*.jpg / publish_guide.txt / script.json
```

## B. 启用 F5 真人克隆音（解决「机器音太假」）

```bash
# 1. 装 F5（较重，含 torch）
pip install f5-tts torch torchaudio

# 2. 参考音库 assets/voices/（已建 female 示例）
#    female.mp3 + female.txt  ← 已就位（复制自你的参考音 mp3）
#    male.mp3   + male.txt    ← 待补；缺失时自动回退 female
#    ⚠️ ref_text 必须和音频逐字一致，建议校准：
python scripts/get_ref_text.py        # whisper 转录，把结果贴回对应 .txt

# 3. 开启 F5 后端
#    .env:  TTS_BACKEND=f5
#           F5_PITCH_STEPS=2.5   # 升调，参考音偏老沉时调高更清亮

# 4. produce 时若 F5 失败（模型没下全/参考音缺失），自动降级 edge-tts，不会中断出片
```

**双后端机制**：`TTS_BACKEND=edge`（默认，机器音）↔ `f5`（克隆音）。
F5 模型模块级缓存，批量生产只加载一次；输出 wav 自动 ffmpeg 转 mp3。

## C. 字幕对齐（解决「字幕和配音对不上」）

字幕时间戳由 **Whisper small 逐字对齐**生成。**对不上的最常见原因：Whisper 没装/没加载成功，
代码降级成「均匀按字数分配」——它不看实际语音，语速一不均就错位。**

排查：
1. 确认 `openai-whisper` 已装、能加载 small 模型（首次会下模型）。
2. 跑 produce 时看日志：若出现 `字幕走【均匀分配】降级` 警告，就是 Whisper 没生效 → 按上面装好。
3. Whisper 正常后仍偏差，可换 `medium` 模型（改 `_get_whisper_model` 的 `load_model("small")`）。
